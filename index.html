Documentation (How to use)

What it does: This page generates three unique random numbers in the chosen range. It's suitable for QuickPick / Lucky Dip style picks. By default the range is 1–49 (UK49s range). Wannts to try more kiwpick 3 generator? Visit star49s kwikpick 3 lunchtime and teatime.

Controls

  • Draw type: Select Lunchtime / Teatime / Random. This is for labeling only and saved in the output metadata.
  • Min / Max: Set your number range. Min must be at least 1 and Max must be at least 3.
  • Generate 3 Numbers: Creates three unique numbers and shows them in big chips.
  • Copy: Copies JSON with numbers and metadata to clipboard.
  • Download JSON: Saves the pick as a small JSON file.

Output format (JSON)

{
  "drawType": "lunchtime",
  "range": [1,49],
  "numbers": [5,23,44],
  "timestamp": "2025-09-09T10:22:33Z"
}
